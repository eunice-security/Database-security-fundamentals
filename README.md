# Database Security Fundamentals

## Overview
This repository contains foundational database and cybersecurity projects focused on secure data handling, SQL database management, and Python database interaction using SQLite.

The project demonstrates core database concepts including table creation, record insertion, structured data management, and database querying techniques commonly used in secure systems administration and cybersecurity environments.
